# Portfolio

Este é o meu portfólio pessoal, onde apresento meus principais projetos de desenvolvimento web, tanto no frontend quanto no backend.
O objetivo é demonstrar minha experiência prática com tecnologias modernas como Node.js, TypeScript, React e Next.js, além de boas práticas de desenvolvimento e arquitetura de software.

O portfólio também funciona como ponto central para conhecer meu trabalho, experiência profissional e formas de contato.

---

## Visão Geral

Neste projeto você encontrará:
- Projetos frontend e backend
- APIs REST desenvolvidas em Node.js
- Integração com bancos de dados
- Uso de TypeScript em toda a stack
- Código organizado e escalável
- Demonstração de boas práticas de desenvolvimento

## Como Executar o Projeto

### Pré-requisitos
- Node.js (versão LTS)

```bash
git clone https://github.com/P3DR0DEV/portfolio.git
cd portfolio
npm install
npm run dev
```

Após iniciar, o projeto estará disponível em ```bash http://localhost:3000```
